1. Download Node ---> https://nodejs.org/en/download/

1. Open command prompt

2. change directory to donbot folder Ex. "cd C:\Users\(your username)\desktop\donbot"

3. Install node. Type ---> npm install npm@latest -g

4. Install PM2. Type ---> npm install pm2@latest -g

5. Change the batch file extension from .bat to .txt

6. Open the batch file in a text editor.

7. Change the cd location of your donbot folder.

8. Save and change the extension back to .bat

9. Run the batch file.
